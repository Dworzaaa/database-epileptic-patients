    This project is being created by the students of bachelor study programme at the Czech Technical University in Prague, Faculty of Information Technology in cooperation with Faculty of Electrical Engineering. The client is the University hospital Motol.

    Project maintained by Dworzaaa Hosted on GitHub Pages — Theme by mattgraham

    Welcome to Database of patients in epilepsy research page.

    Anotation, as well as this whole document, will be continuously updated, depending upon the details that team will get from the client and the supervisor. Project will be released in january 2014. For more details about this project, please check the repo and stay tuned.

    Used technologies and architecture

    Project is now being programmed, and we're in circa 70% of our work. Backend will is running on Java EE + Spring 3 framework. As ORM framework is used Hibernate 4.2.2. As an application server is used Apache Tomcat 7. Frontend is realized via JSP+CSS+js+Twitter bootstrap 3.0 framework. As OS we use Debian 6 and as database MySQL 5.5.27.
